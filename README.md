# APIC-Tool
Automated Point Input Calculator for Mario Kart World tournaments

🏁 A.P.I.C. - Automated Point Input Calculator
Alpha Version (A.0.3)

ℹ️ Overview
Welcome to the A.P.I.C. system, a tool designed to automate the calculation and tracking of tournament scores. This file is optimized for rapid data entry and high precision.

[👉 Click here to use A.P.I.C. : https://docs.google.com/spreadsheets/d/1ltEwDKxLt1ZuO1nYJbuOMcLhI6064NcD/copy?usp=sharing&ouid=114972095422597374043&rtpof=true&sd=true]

🚀 Quick Start Guide

Dashboard: Go to the "12p" tab. This is your main tracking dashboard.

Data Entry: Manually enter player names (automated Discord integration is coming soon). Map names support an active search feature, recognizing both track codes and full names.

Entering Scores & Handling DCs:

Input scores track by track.

CRITICAL FOR DCs: For disconnections, type "DC" in the score box for the affected course.

⚠️ Important Note on Indicators
Columns such as Pline or Ptruefalse are control indicators. If your tournament includes disconnections (resulting in empty cells), these indicators may display unusual results: this is completely normal in this Alpha version. The core score calculation system remains fully operational.

🛠️ Vision & Development
This project is currently in its Alpha Phase. It is not intended to replace Mogi Manager (for users with video capture setups), but rather to offer a complete and efficient alternative for all other tournament organizers.

Roadmap: Future support for 2v2, 3v3, 6v6, and 24p formats. Full automation via VBA scripts is in progress.

Ultimate Goal: To evolve into a standalone application capable of natively integrating tournament management features.
